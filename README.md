## ✨ 功能特点

- 🚀 支持多种AI模型接入(OpenAI、零一万物等)
- 💬 文本对话功能
  - 支持流式输出
  - 支持预设提示词管理
  - Markdown格式过滤
- 🖼️ 图像分析功能
  - 支持图片预览
  - 自定义分析提示词
- ⌨️ 快捷键支持
  - 可自定义全局快捷键
  - 支持划词搜索
  - 支持截图分析
- 🎨 美观的UI界面
  - 无边框窗口设计
  - 自定义主题样式
  - 系统托盘支持
- 🔧 灵活的配置选项
  - API设置
  - 代理配置
  - 模型选择
